# 🌟 Portofolio Pribadi

Selamat datang di portofolio digital saya! 🎉 Ini adalah situs web portofolio modern yang dibuat dengan **React.js**, **Vite**, dan **Tailwind CSS**, tempat saya memamerkan keterampilan, proyek keren, dan pengalaman terbaik saya. Dengan animasi halus, desain responsif, dan efek teks dinamis, proyek ini siap memikat pengunjung! 😎

---

## 🎯 Apa yang Membuat Portofolio Ini Istimewa?

- 📱 **Desain Responsif**: Terlihat memukau di ponsel, tablet, hingga desktop berkat Tailwind CSS.
- ✨ **Animasi Menawan**: Transisi dan gerakan elegan menggunakan **Framer Motion**.
- 🔍 **SEO-Friendly**: Meta tag dinamis dengan **React Helmet** untuk visibilitas optimal di mesin pencari.
- 🛜 **Navigasi Halus**: Berpindah antar bagian dengan efek scrolling lembut via **React Scroll**.
- ⌨️ **Efek Mesin Ketik**: Teks hidup dengan animasi ala mesin ketik dari **React Simple Typewriter**.
- 🖼️ **Ikon Modern**: Koleksi ikon skalabel dan keren dari **React Icons**.

---

## 🛠️ Teknologi yang Digunakan

Proyek ini memanfaatkan teknologi terbaik untuk performa dan estetika:
- **React.js**: Pustaka JavaScript untuk membangun UI interaktif.
- **Vite**: Alat bundling cepat untuk pengembangan modern dan build yang efisien.
- **Tailwind CSS**: Styling cepat dengan pendekatan utility-first.
- **Framer Motion**: Animasi yang memanjakan mata.
- **React Helmet**: Optimasi SEO tingkat lanjut.
- **React Scroll**: Pengalaman navigasi yang mulus.
- **React Simple Typewriter**: Efek teks yang menarik perhatian.
- **React Icons**: Ikon untuk mempercantik antarmuka.

## ⚙️ Cara Menjalankan Proyek

Ingin menjalankan portofolio ini di mesin lokal Anda? Ikuti langkah-langkah berikut:

1. **Kloning Repositori**  
   ```bash
   git clone https://github.com/lewyinn/portofolio.git
   cd portofolio-anda
   ```

2. **Pasang Dependensi**  
   Pastikan [Node.js](https://nodejs.org/) sudah terpasang, lalu jalankan:
   ```bash
   npm install
   ```

3. **Jalankan Server Pengembangan**  
   ```bash
   npm run dev
   ```
   Buka [http://localhost:5173](http://localhost:5173) di browser Anda untuk melihat hasilnya! 🎉 (Port default Vite adalah 5173, tetapi periksa terminal Anda untuk memastikan).

---

## 🚀 Penggunaan

- **Mode Pengembangan**: Gunakan `npm run dev` untuk pengembangan dengan hot reloading super cepat.
- **Build Produksi**: Jalankan `npm run build` untuk menghasilkan versi dioptimalkan di folder `dist`.
- **Pratinjau Build**: Gunakan `npm run preview` untuk melihat build produksi secara lokal.
- **Kustomisasi**: Edit komponen di `src/components/` atau gaya di `src/styles/` untuk menambahkan sentuhan pribadi.

## 📂 Struktur Proyek

```
portofolio-anda/
├── public/            # Aset statis (gambar, favicon, dll.)
├── src/               # Kode sumber proyek
│   ├── assets/        # Aset seperti gambar atau font
│   ├── components/    # Komponen React yang reusable
│   ├── styles/        # File Tailwind CSS dan gaya global
│   ├── App.jsx        # Komponen utama aplikasi
│   ├── main.jsx       # Titik masuk aplikasi
├── index.html         # File HTML utama
├── package.json       # Dependensi dan skrip proyek
├── vite.config.js     # Konfigurasi Vite
├── tailwind.config.js # Konfigurasi Tailwind CSS
└── README.md          # Dokumentasi ini
```
